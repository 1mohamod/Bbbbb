# Discord Admin Bot

بوت تقديم لإدارة ديسكورد باستخدام Slash commands و Modals.

## طريقة التشغيل

1. عدّل ملف `.env.example` وسمه `.env`
2. أضف التوكن و ID البوت فيه
3. ثبت الحزم:
   ```
   npm install
   ```
4. شغّل البوت:
   ```
   npm start
   ```

## الأوامر
- /setup — لإرسال زر التقديم
- /block — لحظر عضو من التقديم
- /unblock — لفك الحظر
